# User Management API

Complete REST API documentation demonstrating endpoint specifications, authentication patterns, and code examples.

**Note**: This is a writing sample for portfolio purposes.

## Quick Start

\`\`\`javascript
const response = await fetch('https://api.example.com/v2/users', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    firstName: 'John',
    lastName: 'Doe'
  })
});

const user = await response.json();
\`\`\`
